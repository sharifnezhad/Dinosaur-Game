# Dinosaur-Game
Design a dinosaur game with an arcade library

**Features of this game:**

- Jumping and bending dinosaur

- Has a sound

- Has day and night mode change

 **Game instructions:**

- Jump the dinosaur with the up key

- Bend the dinosaur with the down key
